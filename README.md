# Student Performance Analysis

## Project Overview
This beginner-level project analyzes student performance to understand how study hours and attendance affect exam results.

The project was implemented using both Excel and Python to learn real-world data analysis workflows.

## Tools Used
- Microsoft Excel
- Python
  - Pandas
  - Matplotlib
- Jupyter Notebook

## Dataset Description
The dataset includes:
- Student ID
- Gender
- Study Hours
- Attendance percentage
- Subject-wise marks (Math, Science, English)
- Final Result (Pass/Fail)

## Analysis Performed
- Data cleaning and validation
- Calculation of average marks per subject
- Pass vs Fail analysis
- Data visualization using:
  - Clustered Column Chart
  - Pie Chart

## Key Insights
- Higher study hours and attendance lead to better performance
- Math has slightly lower average marks compared to other subjects
- Majority of students passed the exam

## Learning Outcome
- Understood end-to-end data analysis workflow
- Practiced Excel for data analysis
- Recreated the same analysis using Python (Pandas & Matplotlib)
